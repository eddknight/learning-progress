Month 1: Foundations (Linux, Python, Networking Basics)



Week 1: Linux Fundamentals



Goal: Learn basic Linux commands, shell scripting, and file management.



Day Topic Activities Time

Mon Intro to Linux Learn directory navigation (cd, ls, pwd), file management (cp, mv, rm, touch). 1 hour

Tue File Permissions Practice chmod, chown, ls -l. Write a Bash script to automate permission changes. 1.5 hours

Wed Text Processing Use grep, awk, sed, cat, less. Practice parsing log files. 1.5 hours

Thu Cron Jobs & Bash Scripting Create a script to back up files and schedule it with cron. 2 hours

Fri Linux Networking Basics Learn ping, netstat, ifconfig, traceroute. Test commands on a VM. 1.5 hours

Sat Mini-Project Write a script to monitor disk space and send alerts (use df, mail). 2 hours

Sun Review & Practice Revisit weak areas, practice all commands, refine your script. 1 hour



Week 2: Python Basics



Goal: Master Python syntax, data structures, and simple scripting.



Day Topic Activities Time

Mon Variables & Data Types Practice basic operations with strings, numbers, and lists. 1 hour

Tue Functions & Loops Write functions for basic tasks (e.g., calculating file sizes). 1.5 hours

Wed File I/O Write a script to read, write, and append to files. 1.5 hours

Thu Error Handling Write a script with try/except blocks (e.g., handling missing files). 1.5 hours

Fri Networking with Python Write a basic port scanner using socket. 2 hours

Sat Mini-Project Create a password generator using random and string libraries. 2 hours

Sun Review & Practice Practice coding challenges on HackerRank or LeetCode. 1 hour



Week 3: Networking Basics



Goal: Learn the fundamentals of networking and basic analysis.



Day Topic Activities Time

Mon Networking Theory Study TCP/IP, OSI model, and subnetting (use YouTube or resources like Cisco NetAcad). 1 hour

Tue Wireshark Basics Capture traffic, identify HTTP and DNS requests. 2 hours

Wed Nmap Basics Scan local networks for open ports/services. 1.5 hours

Thu Network Troubleshooting Use tools like ping, traceroute, and netstat to analyze issues. 1.5 hours

Fri Networking Tools Install and explore Nikto for basic web vulnerability scanning. 1.5 hours

Sat Mini-Project Write a Python script to automate Nmap scans and log results. 2 hours

Sun Review & Practice Practice analyzing packets in Wireshark or scanning with Nmap. 1 hour



Week 4: Integration



Goal: Combine what you’ve learned so far into small projects.



Day Topic Activities Time

Mon Integrating Linux & Python Automate file management tasks (e.g., organizing files by type). 1.5 hours

Tue Networking with Python Extend your port scanner to log results into a file. 2 hours

Wed Basic Web Scanning Use Python and requests to scan for HTTP headers. 1.5 hours

Thu Wireshark Analysis Capture and analyze packets; identify login attempts or HTTP POSTs. 1.5 hours

Fri Mini-Project Build a script to monitor logs for unauthorized login attempts. 2 hours

Sat Progress Review Consolidate notes, refine scripts, and practice weak areas. 1.5 hours

Sun Practice CTFs Start TryHackMe Networking Path labs. 1.5 hours



Months 2-4 (Intermediate and Advanced)



Month 2: Intermediate Skills

• Focus: Write and test simple malware (e.g., keylogger), automate vulnerability scanning, and deepen Linux/Networking knowledge.

• Daily Plan: Alternate between learning and practicing:

• Monday-Friday:

• Write scripts (keylogger, file stealer).

• Automate tasks like Nmap scans with Python.

• Practice with tools like OpenVAS, Nessus, or Wireshark.

• Example: Test a keylogger in a sandboxed VM and analyze its behavior.

• Saturday-Sunday:

• Complete labs on TryHackMe or Hack The Box.



Month 3: Web Application Security

• Focus: Learn web testing (SQL Injection, XSS), build vulnerable web apps, and test them.

• Daily Plan:

• Monday-Friday:

• Develop a Flask/Django app with intentional vulnerabilities (e.g., insecure login).

• Test using Burp Suite and manual methods.

• Write Python scripts to automate attacks (e.g., SQLmap).

• Saturday-Sunday:

• Practice on platforms like OWASP Juice Shop or TryHackMe Web Pentesting Path.



Month 4: Advanced Topics

• Focus: API testing, social engineering, and Security+ prep.

• Daily Plan:

• Monday-Friday:

• Write Python scripts to interact with APIs.

• Simulate phishing with the Social Engineering Toolkit.

• Study Security+ domains and take practice tests.

• Saturday-Sunday:

• Test APIs for vulnerabilities using tools like Burp Suite.

• Work on advanced labs on Hack The Box.
